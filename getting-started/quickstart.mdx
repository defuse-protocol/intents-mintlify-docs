Execute a cross-chain token swap using the 1Click Swap API. You'll swap tokens from one blockchain to another without managing bridges or liquidityâ€”NEAR Intents handles everything automatically.

<Info>
**Prerequisites**: A small amount of tokens on a [supported chain](/resources/chain-support). Start with a test amount ($10-50) for your first swap.
</Info>

## How It Works

Instead of navigating multiple bridges and DEXs, NEAR Intents lets you specify what you want (e.g., "swap 0.1 NEAR for ARB on Arbitrum") and competitive market makers race to give you the best execution:

1. Request a quote for your desired swap
2. Send tokens to a unique deposit address
3. Receive your swapped tokens on the destination chain

If anything goes wrong, your funds are automatically refunded.

## Option 1: Use the SDK (Recommended)

The TypeScript SDK provides the simplest integration path.

```bash
npm install @defuse-protocol/one-click-sdk-typescript
```

```typescript
import { oneClickSwap } from '@defuse-protocol/one-click-sdk-typescript';

const result = await oneClickSwap({
  originAsset: "nep141:wrap.near",
  destinationAsset: "nep141:arb-0x912ce59144191c1204e64559fe8253a0e49e6548.omft.near",
  amount: "100000000000000000000000", // 0.1 NEAR
  recipient: "0xYourArbitrumAddress",
  recipientType: "DESTINATION_CHAIN",
  refundTo: "your-account.near",
  refundType: "INTENTS"
});
```

See the [example repository](https://github.com/near-examples/near-intents-examples) for a complete working implementation with step-by-step code.

**Available SDKs:**

- [Rust](https://github.com/defuse-protocol/one-click-sdk-rs)

## Option 2: Use the REST API

<Steps>
<Step title="Get supported tokens">
Fetch available tokens to find the `assetId` values you'll need.

<CodeGroup>
```bash cURL
curl https://1click.chaindefuser.com/v0/tokens
```

```typescript TypeScript
const response = await fetch('https://1click.chaindefuser.com/v0/tokens');
const tokens = await response.json();
```

```python Python
import requests
response = requests.get('https://1click.chaindefuser.com/v0/tokens')
tokens = response.json()
```
</CodeGroup>

The response includes tokens with their `assetId` in this format:
- NEAR tokens: `nep141:wrap.near`
- Bridged tokens: `nep141:eth-0xdac17f958d2ee523a2206206994597c13d831ec7.omft.near`
</Step>

<Step title="Request a quote">
Request a quote with your swap parameters. Include your JWT token to avoid the 0.1% fee.

<CodeGroup>
```bash cURL
curl -X POST https://1click.chaindefuser.com/v0/quote \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_JWT_TOKEN" \
  -d '{
    "dry": false,
    "swapType": "EXACT_IN",
    "originAsset": "nep141:wrap.near",
    "destinationAsset": "nep141:arb-0x912ce59144191c1204e64559fe8253a0e49e6548.omft.near",
    "amount": "100000000000000000000000",
    "recipient": "0xYourArbitrumAddress",
    "recipientType": "DESTINATION_CHAIN",
    "refundTo": "your-account.near",
    "refundType": "INTENTS"
  }'
```

```typescript TypeScript
const quote = await fetch('https://1click.chaindefuser.com/v0/quote', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  },
  body: JSON.stringify({
    dry: false,
    swapType: 'EXACT_IN',
    originAsset: 'nep141:wrap.near',
    destinationAsset: 'nep141:arb-0x912ce59144191c1204e64559fe8253a0e49e6548.omft.near',
    amount: '100000000000000000000000',
    recipient: '0xYourArbitrumAddress',
    recipientType: 'DESTINATION_CHAIN',
    refundTo: 'your-account.near',
    refundType: 'INTENTS'
  })
});
const result = await quote.json();
```

```python Python
import requests

quote = requests.post(
  'https://1click.chaindefuser.com/v0/quote',
  headers={
    'Content-Type': 'application/json',
    'Authorization': 'Bearer YOUR_JWT_TOKEN'
  },
  json={
    'dry': False,
    'swapType': 'EXACT_IN',
    'originAsset': 'nep141:wrap.near',
    'destinationAsset': 'nep141:arb-0x912ce59144191c1204e64559fe8253a0e49e6548.omft.near',
    'amount': '100000000000000000000000',
    'recipient': '0xYourArbitrumAddress',
    'recipientType': 'DESTINATION_CHAIN',
    'refundTo': 'your-account.near',
    'refundType': 'INTENTS'
  }
)
result = quote.json()
```
</CodeGroup>

<Tip>
Set `dry: true` to preview pricing without generating a deposit address. Set `dry: false` when ready to execute.
</Tip>

**Key parameters:**

| Parameter | Description |
|-----------|-------------|
| `swapType` | `EXACT_IN` (specify input amount) or `EXACT_OUT` (specify output amount) |
| `originAsset` | Source token `assetId` from the tokens endpoint |
| `destinationAsset` | Target token `assetId` from the tokens endpoint |
| `amount` | Amount in smallest unit (wei, yoctoNEAR, etc.) |
| `recipient` | Address to receive swapped tokens |
| `recipientType` | `DESTINATION_CHAIN` (native address) or `INTENTS` (NEAR account) |
| `refundTo` | Address for refunds if swap fails |
| `refundType` | `DESTINATION_CHAIN` or `INTENTS` |
</Step>

<Step title="Send tokens">
Transfer tokens to the `depositAddress` from the quote response. The swap begins automatically upon receipt.

Save the deposit address and your transaction hash for tracking.
</Step>

<Step title="Submit transaction hash (optional)">
Speed up processing by notifying 1Click of your deposit.

<CodeGroup>
```bash cURL
curl -X POST https://1click.chaindefuser.com/v0/deposit/submit \
  -H "Content-Type: application/json" \
  -d '{
    "depositAddress": "address-from-quote-response",
    "txHash": "0xYourTransactionHash"
  }'
```

```typescript TypeScript
await fetch('https://1click.chaindefuser.com/v0/deposit/submit', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({
    depositAddress: 'address-from-quote-response',
    txHash: '0xYourTransactionHash'
  })
});
```

```python Python
requests.post(
  'https://1click.chaindefuser.com/v0/deposit/submit',
  json={
    'depositAddress': 'address-from-quote-response',
    'txHash': '0xYourTransactionHash'
  }
)
```
</CodeGroup>
</Step>

<Step title="Monitor status">
Check swap progress using the deposit address.

<CodeGroup>
```bash cURL
curl "https://1click.chaindefuser.com/v0/status?depositAddress=your-deposit-address"
```

```typescript TypeScript
const status = await fetch(
  'https://1click.chaindefuser.com/v0/status?depositAddress=your-deposit-address'
);
const result = await status.json();
```

```python Python
status = requests.get(
  'https://1click.chaindefuser.com/v0/status',
  params={'depositAddress': 'your-deposit-address'}
)
```
</CodeGroup>

**Swap statuses:**

| Status | Description |
|--------|-------------|
| `PENDING_DEPOSIT` | Awaiting your token deposit |
| `PROCESSING` | Deposit detected, swap being executed |
| `SUCCESS` | Tokens delivered to destination address |
| `INCOMPLETE_DEPOSIT` | Deposit below required amount |
| `REFUNDED` | Swap failed, funds returned to refund address |
| `FAILED` | Swap encountered an error |
</Step>
</Steps>

Get a JWT token from the [Partner Dashboard](https://partners.near-intents.org/) to avoid the 0.1% (10 basis points) fee. Include it in your requests:

```
Authorization: Bearer YOUR_JWT_TOKEN
```

See the [Authentication guide](/api-reference/authentication) for details.

## Troubleshooting

<AccordionGroup>
<Accordion title="Swap taking a long time">
  - Check the blockchain explorer for your deposit transaction
  - Verify the deposit address matches the quote response
  - Allow up to 15 minutes for cross-chain processing
  - Use the status endpoint to check current state
</Accordion>

<Accordion title="Swap failed or was refunded">
  - Verify you sent the exact amount specified in the quote
  - Check your refund address for returned funds
  - Ensure your destination address format is correct for the target chain
  - Request a new quote and try again
</Accordion>
